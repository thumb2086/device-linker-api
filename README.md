# Device Linker API

子熙幣娛樂大廳與鏈上結算 API，包含賭場遊戲、排行榜、金融市場、錢包、VIP 等級與後台工具。

## 目前內容
- 錢包登入、快速登入、維護模式與管理員後台
- 多款遊戲 API 與前端頁面
- 排行榜、歷史榜王、連冠顯示
- 金融市場模擬交易、銀行、貸款、期貨與快取快照
- 全站設定、BGM、彈幕與聊天室

## 主要頁面
- `/`：大廳首頁
- `/games/casino.html`：遊戲列表
- `/games/leaderboard.html`：排行榜
- `/games/market.html`：金融市場
- `/games/wallet.html`：錢包
- `/games/vip.html`：VIP 中心
- `/games/support.html`：支援與公告
- `/games/admin-tools.html`：管理工具

## 金融市場
- 正式 API：`POST /api/market`
- 相容轉接：`POST /api/market-sim`
- 目前支援：
  - 現股買入 / 賣出
  - 快速交易佇列
  - 期貨開倉 / 平倉
  - 銀行存入 / 提領
  - 貸款 / 還款
  - 市場快取快照、板塊趨勢與大盤指數

- 中高段等級已合併整理為嚴格的 `2 / 5 / 10` 升級門檻節奏，移除原本穿插的 `3 / 8 / 12 / 15 / 25` 等級距。

- 目前最高單注上限為 `10 兆` 子熙幣。
