# Reward Items Guide

更新日期：`2026-03-09`

## Buff 道具

### `profit_boost_small` 小型獲利翻倍卡
- 效果：15 分鐘內淨盈利 `x2`
- 上限：總加成最多 `20,000,000`
- 適用：單人遊戲結算

### `profit_boost_large` 大型獲利翻倍卡
- 效果：30 分鐘內淨盈利 `x2`
- 上限：總加成最多 `500,000,000`
- 適用：單人遊戲結算

### `loss_shield_single` 單次免損護盾
- 效果：失敗時返還本金 `1` 次
- 限制：按次數保護，不設金額上限
- 適用：單人遊戲結算

### `loss_shield_triple` 三次免損護盾
- 效果：失敗時返還本金 `3` 次
- 限制：按次數保護，不設金額上限
- 適用：單人遊戲結算

### `loss_shield_timed` 15 分鐘免損護盾
- 效果：15 分鐘內最多保護 `3` 次失敗
- 限制：按次數保護，不設金額上限
- 適用：單人遊戲結算

### `luck_boost` 幸運增幅卡
- 效果：提高寶箱高稀有獎勵權重
- 限制：不直接提高遊戲勝率
- 適用：寶箱、補給箱、額外掉落

## 寶箱

### `rare_chest` 稀有寶箱
- 可得：子熙幣、基礎 Buff、頭像、稱號

### `super_rare_chest` 超稀有寶箱
- 可得：進階 Buff、稀有外觀、稱號

### `epic_chest` 史詩寶箱
- 可得：高價 Buff、史詩外觀、稱號

### `mythic_chest` 神話寶箱
- 可得：高價值道具、神話稱號、外觀

### `legendary_chest` 傳奇寶箱
- 可得：頂級 Buff、傳奇稱號、限定外觀

## 補給箱

### `basic_key` 普通鑰匙補給箱
- 可得：基礎資源、Buff 或額外寶箱

### `advanced_key` 高級鑰匙補給箱
- 可得：進階 Buff、稱號或高價值資源

### `master_key` 萬用鑰匙補給箱
- 可得：高階稱號、傳奇資源與頂級 Buff
